# Engenharia-de-Software-Project
Projeto para uma cidade inteligente requisitado na matéria de Tópicos em Engenharia de Software

## Definindo o Problema 

Constantemente com o crescimento das cidades, pequenos problemas se tornam maiores com o passar do tempo, o avanço acelerado de estruturas físicas de cidades desenvolvidas acaba ofuscando aquela parte menos visível socialmente, na maioria dos casos a parte menos desenvolvida da cidade, o subúrbio. 
	O problema em foco é o lixo, quanto mais afastado da cidade, mais escasso é o tratamento com isso acaba obrigando a população de áreas rurais a descartarem o lixo de forma inconsequente.
